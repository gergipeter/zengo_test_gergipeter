<h2>🛠️ Installation Steps:</h2>

    git clone https://github.com/gergipeter/zengo_test_gergipeter.git
    cd zengo_test_gergipeter
    composer update
    cp .env.example .env
    php artisan migrate
    php artisan db:seed --class=CountiesTableSeeder

    php artisan serve
    npm install
    npm run dev
